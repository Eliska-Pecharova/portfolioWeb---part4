    Answer Generator

    An interactive web application for generating random answers. The project combines simple logic in TypeScript with a modern visual style — including an animated background with question marks, a clear interface, and buttons for generating and resetting. Suitable for fun use, testing, or as part of a portfolio.

    This project was created as my first larger web application, when I was just getting acquainted with JavaScript and TypeScript. I gradually mastered working with the DOM, events, and type safety, and created a functional response generator with clean code and a clear structure.

    The result is a modern, responsive website with interactive elements, a stylish background, and intuitive controls. The project serves not only as a demonstration of technical skills, but also as a playful tool for brainstorming, presentations, or entertainment.
