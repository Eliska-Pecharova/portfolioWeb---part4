    Filtering Harry Potter characters

    An interactive web application for filtering characters from the Harry Potter world. The project combines simple logic in TypeScript with a modern visual style – including a thematic background, a clear structure and intuitive controls. The user can select characters by house, gender or role in the story.

    I created this project as an extension of my skills in working with data, DOM, and events. I focused on clean code, responsive design, and user friendliness. The result is a playful yet technically sophisticated application that combines a fun theme with practical filtering.
