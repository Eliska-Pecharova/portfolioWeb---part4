    Gardening Pod Skalkou – redesign

    This project was created at the request of a client who needed to improve their existing gardening website. The goal was to preserve the original content, but significantly improve the visuals, user-friendliness and functionality.

    This was my first major web project, which I implemented at a time when I had not yet worked with JavaScript or TypeScript.

    The result is a modern, responsive website with a clear structure, interactive elements and optimization for mobile devices.
