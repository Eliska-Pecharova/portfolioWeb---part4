    Travel blog

    This project is my very first output as part of the Engeto.cz course. The goal was to create a simple static website — a travel blog — using pure HTML and CSS without using JavaScript or backend technologies.

    The project serves as a demonstration of basic content structuring, working with typography, color palette, and responsive design. Despite its simplicity, it represents an important step in my development as a frontend developer.

    GitHub

    Introduction to the site

    The site is not just a blog, but a full-fledged travel portal that combines personal stories, practical advice, inspiration and services. It is designed so that the visitor does not only dream of traveling, but also starts planning right away.

    Selection of parts of the page

    Sample sections describing the "About Me" and "What We Do" sections plus a photo gallery.

    Contact and comments section

    The "Contact" section gives you the opportunity to contact an assistant who will help you put together a tailor-made trip. The "Reviews" section contains authentic experiences from clients that confirm the quality of the services.
