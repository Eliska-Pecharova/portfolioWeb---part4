    Netflix clone

    This project was created as the second task within the Engeto.cz course. The goal was to create a visual clone of the Netflix homepage with an attractive design, clean code, and interactive elements.

    The project shows my experience in frontend development, but also the limits of the static approach — for example, the absence of a direct backend or user login. Despite the absence of these elements, it demonstrates a visual and functional similarity to the original website.
